# Knowledge_node
This is a knowledge base app.

## Technologies
 * Node.js
 * Express
 * Express Messages, Session, Connect Flash & Validation
 * MongoDB & Mongoose
 * Pug Templating
 * Passport.js Authentication
 * BCrypt Hashing

Usage
Installation
Install the dependencies

```sh
$ npm install
```
Run app

```sh
$ npm start
```
